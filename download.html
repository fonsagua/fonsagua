Descarga

Como una aplicación completa

gvSIG Fonsagua se distribuye como un archivo zip que contiene gvSIG 1.1.2 portátil y un conjunto de aplicaciones, la mayoría desarrolladas por CartoLab, para diseñar redes WatSan. Cómo instalarlo:

  • Descarga este archivo zip [100MB].
  • Descomprímelo donde quieras. Incluso en un dispositivo USB. No es necesaria ninguna instalación.
  • Navega hasta el directorio bin y ejecuta gvSIG.exe.

Conjunto de datos de prueba

Para utilizar gvSIG Fonsagua necesitas una estructura de información predefinida. Puedes utilizar un conjunto de datos que hemos creado para realizar tus propias pruebas en la aplicación. Esta información no es real y ha sido creada exclusivamente con fines de prueba. Para utilizarlo, descomprime los datos donde quieras y establece correctamente la ruta a los mismos presionando el botón de configuración dentro de la aplicación.

  • Descarga el dataset o conjunto de datos de prueba.
