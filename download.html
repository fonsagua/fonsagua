Descarga

Como una aplicación completa

gvSIG Fonsagua se distribuye como un archivo comprimido que contiene todo lo necesario para ejecutar el software. Cómo instalarlo:

  • Descarga el fichero adecuado para tu sistema operativo desde nuestra página de releases
  • Descomprímelo donde quieras. Incluso en un dispositivo USB. No es necesaria ninguna instalación.
  • Navega hasta el directorio bin y ejecuta gvSIG.exe si estás en Windows o gvSIG.sh si estás en Linux.

Conjunto de datos de prueba

Para utilizar gvSIG Fonsagua necesitas una estructura de información predefinida. El fichero fonsagua.sqlite dentro del directorio cfg es una base de datos ya preparada para trabajar con gvSIG Fonsagua. Además ha sido cargada con Cartografía Base adaptada a los municipios correspondientes a NASMAR en la zona Sur de Honduras. Puedes preparar tus propios datos y cargarlos en la aplicación o contactarnos si necesitas ayuda.
