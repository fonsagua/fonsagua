Preguntas frecuentes

A continuación mostramos algunas de las preguntas más frecuentes sobre gvSIG Fonsagua. Si tienes alguna duda no contemplada en esta sección, por favor, ¡ponte en contacto con nosotros!

  1. ¿Puedo utilizar la aplicación en cualquier situación?
  2. ¿Pretende esta aplicación reemplazar a EPANET?
  3. ¿Puedo utilizar gvSIG Fonsagua para recopilar información con otras metodologías?
  4. ¿Contempláis la continuación del desarrollo del software?

¿Puedo utilizar la aplicación en cualquier situación?

Si, puedes. Por favor, decarga el software y ayúdanos a mejorarlo!

La apilcación se encuentra bajo licencia GPL3, una Licencia de Software Libre que permite a los usuarios utilizar el programa para cualquier propósito, analizar o cambiar el código fuente, redistribuir copias de la aplicación y de tus propias versiones. Además de esto, es gratuito.

¿Pretende gvSIG Fonsagua reemplazar a EPANET?

No, tiene diferentes objetivos. EPANET es una buena herramienta para diseñar redes hidrológicas antes de ejecutarlas. Proporciona una gran capacidad de control a los usuarios y tiene una curva de aprendizaje media-alta. gvSIG Fonsagua tiene la intención de ayudar al usuario a planificar de manera fácil y rápida varias propuestas y ver cómo funcionan.

¿Puedo utilizar gvSIG Fonsagua para recopilar información con otras metodologías?

Por supuesto, puedes, pero probablemente necesitarás adaptar el código fuente de gvSIG Fonsagua.

Actualmente, la estructura del software y los formularios están diseñados exclusivamente para la metodología utilizada por ISF en sus proyectos para recolectar información de tipo social (ver PDF) y técnica (ver PDF). Estamos trabajando para conseguir dar más poder a los usuarios y ayudarlos a diseñar sus propios formularios de una manera sencila, que incluya otras metodologías como la utilizada por WaterPointMapping, por ejemplo (ver hoja de ruta a continuación).

Si quieres unirte a nosotros para lograr este objetivo, ¡eres bienvenido!

¿Continuaréis con el desarrollo de gvSIG Fonsagua?

Si. En concreto tenemos un montón de cosas en mente sobre las que nos gustaría trabajar. Por ejemplo:

  • Adaptar gvSIG Fonsagua para trabajar con otras metodologías en la recopilación de datos y permitir que el usuario diseñe sus propios formularios.
  • Generación de informes más potentes, incluidos mapas y gráficos.
  • Incluir nuevas herramientas CAD avanzadas para ayudar al usuario a diseñar alternativas.
  • Desarrollar nuevos algoritmos de análisis para obtener información sobre los datos utilizados.
  • Continuar actualizando la plataforma gvSIG en las nuevas versiones de este software.
  • ¡Y mucho más!

Si tienes interés especial en alguna de estas características, deseas ayudar o mantenerte actualizado respecto a las próximas novedades de gvSIG Fonsagua, consulta cómo puedes unirte al equipo.
