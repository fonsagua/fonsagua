¿Cómo funciona gvSIG Fonsagua?

gvSIG Fonsagua es una aplicación versátil para planificar aspectos relacionados con la gestión del agua y el acceso a la misma en un área determinada.

En otras palabras, gvSIG Fonsagua integra en una herramienta única los recursos necesarios para que los usuarios recopilen, analicen y obtengan información en el contexto de los programas de gestión del agua y del saneamiento.

Partiendo del software gvSIG como plataforma para desarrollar y agregar nuevas funcionalidades, los usuarios de la nueva aplicación, denominada gvSIG Fonsagua, han mejorado sustancialmente la manera en que llevan a cabo su trabajo.

Esta aplicación permite

  1. Importar Puntos de referencia GPS de una manera automatizada.
  2. Introducir datos de campo a través de formularios personalizados.
  3. Diseñar y calcular redes Hidráulicas.
  4. Imprimir informes y utilizar consultas integradas para acceder a la base de datos.
  5. Priorización de alternativas y presupuesto..
  6. Utilizar todas las capacidades propias de una aplicación GIS.

1. Importar Puntos de referencia GPS de una manera automatizada.

Mientras recopila información de campo y mediante una herramienta de GPS, el usuario recopila los puntos de interés en cada comunidad. La aplicación permite al usuario importar de manera automática esos puntos a la base de datos.

2. Introducir datos de campo a través de formularios personalizados.

El usuario introduce la información en la base de datos a través de formularios personalizados. Estos formularios han sido definidos teniendo en cuenta la metodología utilizada para recopilar información de campo a lo largo del proyecto con consideraciones sociales (ver PDF) y técnicas (ver PDF).

VER VIDEOTUTORIAL: introducción de datos.

3. Diseñar y calcular redes Hidráulicas.

El usuario puede diseñar y planificar propuestas de agua y saneamiento utilizando gvSIG Fonsagua, basado en criterios técnicos y sostenibles.

El módulo hidráulico hace los cálculos mediante EPANET simplificando la introducción y cálculo de los parámetros mediante el uso de wizards, comprobaciones topológicas, sugerencias sobre errores comunes y otras ayudas.

VER VIDEOTUTORIALES: sistema de bombeo – red de distribución.

4. Imprimir informes y utilizar consultas integradas para acceder a la base de datos.

Una vez introducidos los datos, es posible generar informes teniendo en cuenta el estado actual de la comunidad o su estado después de aplicar una de las alternativas. Además, el usuario puede consultar la base de datos con sentencias predefinidas y exportar los datos a otros formatos.

Los informes se generan automáticamente utilizando bibliotecas Java estándar como iText y OpenCSV.

5. Priorización de alternativas y presupuesto.

La aplicación permite generar un presupuesto inicial del costo de cada alternativa, así como priorizar por cada comunidad las alternativas disponibles en base a criterios económicos, técnicos y sociales.

6. Utilizar todas las capacidades propias de una aplicación GIS.

gvSIG es un Sistema de Información Geográfica, es decir, una aplicación de escritorio diseñada para capturar, almacenar, manejar, analizar e implementar cualquier tipo de información geográfica referenciada para resolver problemas complejos de administración y planificación. gvSIG es conocido por tener una interfaz fácil de usar, pudiendo acceder a los formatos más comunes, tanto vectoriales como raster. Además, cuenta con una amplia gama de herramientas para trabajar con información geográfica (herramientas de consulta, creación de diseños, geoprocesamiento, redes, etc.).
