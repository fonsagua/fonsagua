Download

As a full application

gvSIG Fonsagua is distributed as a zip file that contains gvSIG 1.1.2 portable and a set of applications, most of then developed by CartoLab, to design WatSan networks. How to install it:

  • Download this zip file [100MB].
  • Unzip it where you want. Even in a USB drive. No installation it’s really needed.
  • Go to the folder bin, and run gvSIG.exe.

A test dataset

In order to use gvSIG Fonsagua you need a predefined structure of information. You can use a dataset we have built to make your own tests of the application. This data is unreal and just for testing purposes. To use it unzip the data where you want and inside the application configure the path to it pressing the configure button.

  • Download the dataset.
